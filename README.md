## Project Description
This project focuses on performing Exploratory Data Analysis (EDA) on Swiggy restaurant data to understand pricing patterns, customer rating behavior, and restaurant popularity trends.

## Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Key Steps Performed
- Data cleaning and preprocessing  
- Handling missing values and duplicates  
- Univariate and Bivariate analysis  
- Feature engineering (price per rating, popularity score)  
- Data visualization using charts and plots  

## Key Insights
- Identified pricing distribution across restaurants  
- Analyzed relationship between price and customer ratings  
- Observed popularity trends based on ratings and pricing  

## Outcome
The analysis provided meaningful insights that can support data-driven decision making in food delivery platforms.

## Project Type
Exploratory Data Analysis (EDA)

# Swiggy Data Analysis Project

This project performs Exploratory Data Analysis (EDA) on a Swiggy restaurant dataset using Python.
The goal is to understand pricing, ratings, and restaurant trends through data analysis and visualization.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Work Done
- Loaded and inspected the dataset
- Performed data cleaning and handling
- Calculated basic descriptive statistics
- Conducted exploratory data analysis (EDA)
- Created visualizations for price, ratings, and categories
- Derived simple insights from the data

## Key Insights
- Most restaurants fall within a mid-price range
- Higher-rated restaurants are more common in certain categories
- Data shows clear patterns in pricing and ratings
